# **The Impact of Economics on Malaysia Ringgit.**

To explore how changes in economic indicators affect the value of the Malaysian Ringgit, offering significant new insights on the relationship between economic indicator and currency volatility.

### Datasets:

| Data | Description |
|------|------------|
| Currency Exchange Rates | The exchange rate USD-MYR | 
| GDP Income approach | Malaysia's total income earned |
| GDP GDP Production and Expenditure approach | GDP measured by total output produced and total spending |
| Inflation Rate | Malaysia's inflation rate |
| Interest Rate | Malaysia's interest rate |
| Trade Balance | Difference between Malaysia’s exports and imports |

### Tools Used:
1. Microsoft Excel
2. Python Programming (For Data Cleaning and Transformation)
3. Power Bi (Data Visualization)
4. Microsoft Word (Reporting)
